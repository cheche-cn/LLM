# 八、常见问题 FAQ

## Q1: 如何选择合适的模型？
根据场景、预算、部署方式综合考虑。参考上面的选型指南。

## Q2: 开源模型和闭源模型哪个好？
- **闭源**：性能更强，稳定性好，但成本高
- **开源**：可定制，数据隐私，但需要技术能力

## Q3: 如何降低 API 成本？
1. 使用批量处理（50% 折扣）
2. 选择性价比高的模型（DeepSeek、Qwen）
3. 缓存常见问题答案
4. 优化提示词，减少 token 消耗

## Q4: 本地部署需要什么硬件？
- **7B 模型**：16GB 显存（FP16）/ 8GB（INT4）
- **13B 模型**：26GB 显存（FP16）/ 12GB（INT4）
- **70B 模型**：140GB 显存（FP16）/ 40GB（INT4）

## Q5: 如何评估模型性能？
1. **基准测试**：MMLU、HumanEval、GSM8K
2. **实际测试**：用真实业务数据测试
3. **A/B 测试**：对比不同模型效果

## Q6: 模型会泄露训练数据吗？
有可能，尤其是记忆性强的模型。建议：
- 不要输入敏感信息
- 使用本地部署处理敏感数据
- 选择有隐私保护的服务商

## Q7: 如何防止模型幻觉？
1. 使用 RAG（检索增强生成）
2. 要求模型引用来源
3. 多次生成并对比
4. 选择低幻觉率模型（Claude）

## Q8: 中文模型和英文模型如何选择？
- **中文场景**：优先国产模型（文心、通义、智谱）
- **英文场景**：优先国际模型（GPT、Claude、Gemini）
- **双语场景**：Qwen、GPT-4o
